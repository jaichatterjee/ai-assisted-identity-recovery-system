# AI-Assisted Identity Recovery System

## Overview

The AI-Assisted Identity Recovery System is a Zero Trust aligned architecture designed to orchestrate password reset, MFA recovery, and identity failure handling through natural language interactions.

The architecture integrates:

- Microsoft Copilot Studio
- Microsoft Entra ID
- Conditional Access
- Microsoft Intune
- Microsoft Sentinel
- Power Automate
- ServiceNow

---

## Core Principle

AI is not a trust authority.

AI performs:

- intent detection
- workflow orchestration
- decision routing

Authentication and authorization remain controlled by:

- Entra ID
- Conditional Access
- Identity Protection

---

## Objectives

- Reduce helpdesk dependency
- Support secure password recovery
- Enable risk-based authentication
- Improve user experience
- Contain compromised identity scenarios
