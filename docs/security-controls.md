# Security Controls

## Mandatory Controls

- Conditional Access
- MFA number matching
- Intune device compliance
- Session monitoring
- Audit logging
- Identity Protection

---

## Explicitly Disallowed

❌ AI performing authentication

❌ Password reset without risk evaluation

❌ MFA reset without containment

❌ Weak fallback methods without governance

---

## Security Principle

Trust signals must be continuously evaluated.
